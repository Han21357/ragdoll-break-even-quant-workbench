3 第三步：配置行情数据源

WyckoffAgent 内置五级降级数据链路，确保至少有一种方式能获取到行情：

tickflow(★首选) → tushare → akshare → baostock → efinance
TickFlow ★ 首选高质量 A股/港股/美股日线+实时行情+分钟K线+财务数据。免费模式可用历史日K线，完整模式需 API Key。
Tushare需注册获取 Token（免费额度有限）
akshare完全免费开源，无需 API Key，已自动安装
baostock免费证券数据，已自动安装
efinance免费金融数据接口，已自动安装
推荐做法
零配置即可使用 — akshare/baostock/efinance 已随 pip 安装就绪，无需额外配置即可拉取 A 股历史日K线。先用免费源体验，后续需要实时数据和分钟K线时再注册 TickFlow。

如需 TickFlow 实时数据：注册后配置 API Key：

export TICKFLOW_API_KEY="your-tickflow-key"
4 第四步（可选）：Supabase 云端同步

不配置 Supabase 完全可用。所有数据默认存本地 SQLite（~/.wyckoff/wyckoff.db）。

配置 Supabase 后可实现多设备持仓同步、信号云端共享等高级功能：

export SUPABASE_URL="https://xxx.supabase.co"
export SUPABASE_KEY="your-anon-key"

💡 也可加入知识星球（¥518/年）使用作者维护的共享基础设施，省去自己搭建运维成本。

5 第五步：启动使用

快速启动脚本已生成：./wyckoff.sh

常用启动方式

./wyckoff.sh启动 CLI 对话（TUI 终端界面）
./wyckoff.sh dashboard启动本地可视化面板（浏览器打开）
./wyckoff.sh model list查看已配置的模型列表
./wyckoff.sh screen直接执行全市场漏斗筛选
./wyckoff.sh report生成 AI 深度研报
./wyckoff.sh backtest策略历史回测
./wyckoff.sh session list查看历史对话会话
./wyckoff.sh config tickflow <key>配置 TickFlow API Key

CLI 内置快捷命令

/model /screen /checkup /report /backtest /daily /fork
